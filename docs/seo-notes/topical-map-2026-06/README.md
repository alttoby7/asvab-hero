## Known gaps / caveats
- **Calculator pillar undercounts.** "asvab calculator" (~600/mo) + AFQT/GT/line-score
  calculator variants merged into score clusters during auto-grouping; I surfaced a
  flagship calculator page manually but it has no enrichment row, so the pillar shows ~70/mo.
  Real opportunity here is larger — treat the calculator hub as a priority despite the number.
- **Assembling Objects (AO) subtest** missing from competitor harvest — add manually.
- **dummies.com metrics were fabricated** by a sub-agent; its *keyword strings* were kept
  (valid topics) but all volumes come from real Ahrefs enrichment, not dummies.
- Ahrefs Lite plan caps top-pages at ~10 rows/site, so this is a focused (not exhaustive) map.
  Re-run at higher depth or add iteration-2 snowball for long-tail.
