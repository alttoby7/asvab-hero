# Counselor / educator outreach templates

For the DR / link-building program (see `dr-improvement-plan-2026-06.md`).
**Deliverability rules are binding** — send 1:1 from a Google Workspace mailbox
(`info@`/`trish@asvabhero.com`), **<=8-10 personalized/week**, ONE follow-up,
NO cold-email tools on the root domain (`docs/outreach-deliverability.md`).

These are skeletons. Every send MUST be personalized: name the recipient's
specific page and one real detail from it. Generic sends train spam filters and
get ignored. Lead with the free asset, not a request to "link my homepage."

---
