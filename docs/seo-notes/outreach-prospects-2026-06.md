# ASVAB Hero — Outreach prospect list

_Built 2026-06-17 from Ahrefs: the do-follow, non-spam school/library/career pages
that **already link competitor `asvabtutor.com`** (DR 25 peer). These are the warmest
possible targets — each has already chosen to put a free ASVAB tool on a resource
page, so the ask is "add ours as an additional free, no-signup option," not "please
link my site." Pairs with `dr-improvement-plan-2026-06.md` + `counselor-outreach-templates.md`._

**Deliverability rules are binding:** 1:1 from a Workspace mailbox (`trish@`/`info@asvabhero.com`),
**≤8–10 personalized/week**, ONE follow-up, NO cold-email tools on the root domain
(`docs/outreach-deliverability.md`). Personalize every send: name their exact page +
one real detail. Lead with the free asset (calculator / `/counselor-resources` / one-line embed).

**Asset to pitch by page type:** counseling/career page → `/counselor-resources` (citeable) + `/afqt-calculator`; JROTC/practice page → the `/embed` one-line calculator; "study guide" link → a `/study/...` guide or the calculator.

Status legend: `todo` (find contact) · `queued` (contact found, not sent) · `sent` · `replied` · `linked` · `dead`.
